# VERDICTIF PRO v5.4

Test psychométrique interactif en ligne basé sur le protocole **OMEGA v1.5**, avec analyse de personnalité, tests cognitifs, mesures comportementales et génération d’un rapport détaillé.

## Démonstration

Site GitHub Pages :

**https://titi211179.github.io/venez-tester-votre-personnalite-en-profondeur-/**

> Si le dépôt porte encore son ancien nom, remplace ce lien par l’adresse affichée dans **Settings → Pages**.

## Fonctionnalités

- Profil Jung / MBTI
- Modèle Big Five — OCEAN
- Profil DISC
- Intelligence émotionnelle
- Growth Mindset
- Tolérance à l’ambiguïté
- Fondements moraux
- Tests cognitifs : N-Back, Raven et Stroop
- Analyse des temps de réponse
- Tests d’attention et gestion des distractions
- Indicateurs de cohérence et de fiabilité
- Analyse d’adéquation professionnelle
- Génération et téléchargement d’un rapport HTML détaillé
- Impression ou export PDF du rapport

## Durée

La durée estimée du parcours complet est d’environ **75 minutes**.

## Technologies

Le projet fonctionne dans un seul fichier web :

- HTML5
- CSS3
- JavaScript natif
- Google Fonts
- Aucun framework
- Aucun serveur applicatif requis

## Installation locale

1. Télécharge le dépôt.
2. Vérifie que la page principale s’appelle :

```text
index.html
```

3. Double-clique sur `index.html` pour ouvrir le test dans ton navigateur.

## Publication avec GitHub Pages

1. Ouvre le dépôt GitHub.
2. Va dans **Settings → Pages**.
3. Choisis :
   - **Source** : Deploy from a branch
   - **Branch** : `principal`
   - **Folder** : `/(root)`
4. Clique sur **Save**.
5. Attends quelques minutes, puis clique sur **Visit site**.

Structure recommandée :

```text
/
├── index.html
└── README.md
```

## Mise à jour du site

Après modification de `index.html` :

1. Valide avec **Commit changes**.
2. Attends une à trois minutes.
3. Recharge le site avec `Ctrl + F5`.

## Confidentialité

Le fichier actuel exécute le test dans le navigateur et ne contient pas d’appel réseau destiné à transmettre les réponses vers un serveur externe.

Les résultats téléchargés peuvent toutefois contenir des données personnelles. Leur collecte, conservation et utilisation doivent respecter le RGPD et les obligations applicables à l’organisme qui utilise l’outil.

## Avertissement

VERDICTIF PRO est un **outil d’aide à l’évaluation et à la décision**. Il ne constitue pas un diagnostic médical, psychologique ou psychiatrique, et ne doit pas être utilisé comme seul critère de recrutement, d’exclusion ou de décision professionnelle.
